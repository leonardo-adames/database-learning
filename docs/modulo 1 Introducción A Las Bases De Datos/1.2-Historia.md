##  **Historia De las bases de datos**

La historia de las bases de datos es una evolución desde **fichas perforadas hasta la nube**, comenzando con **Herman Hollerith** **(fichas)en 1884**, pasando a **sistemas jerárquicos y de red** en los 60s **(IMS, CODASYL)**, y revolucionándose en **1970 con Edgar Codd y el modelo relacional**, que dio lugar a **SQL y sistemas como Oracle en los 80s**. **Los 90s** vieron las **bases de datos orientadas a objetos** y herramientas como **Access**, mientras que el siglo XXI trae la explosión de **NoSQL (MongoDB, Cassandra)** y **soluciones en la nube**, enfocándose en **análisis e IA.**

<br>

## **Orígenes y Primeros Pasos (Siglo XIX - 1960s)**

**1884:** Herman Hollerith inventa la máquina tabuladora con fichas perforadas para procesar el censo de EE. UU..

**1950s-1960s:** Auge de las cintas magnéticas y los discos, dando lugar a los primeros sistemas de archivos planos y la necesidad de ordenar datos.

**1960s:** Nacen las bases de datos jerárquicas (IMS) y de red (CODASYL, Charles Bachman), buscando reducir redundancia con estructuras de árbol y redes.

<br>

## **La Revolución Relacional (1970s - 1980s)**

**1970:** Edgar F. Codd de IBM publica su modelo relacional, introduciendo tablas, filas y columnas, fundamental para la integridad de datos.

**1980s:** Surge SQL (Structured Query Language) como estándar, y sistemas como DB2 y Oracle dominan el mercado relacional, ofreciendo gran integridad.

<br>

## **Diversificación y Web (1990s - 2000s)**

**1990s:** Auge de las bases de datos orientadas a objetos (POO) para datos complejos y herramientas gráficas como Access.

**2000s:** Aparición de bases de datos NoSQL (NoSQL), como MongoDB y Cassandra, para manejar grandes volúmenes de datos no estructurados en la web.
**

<br>

## **Era Actual**

**Hoy:** Las bases de datos son la columna vertebral de internet, la analítica de datos, la inteligencia artificial y la computación en la nube (Cloud Computing).
