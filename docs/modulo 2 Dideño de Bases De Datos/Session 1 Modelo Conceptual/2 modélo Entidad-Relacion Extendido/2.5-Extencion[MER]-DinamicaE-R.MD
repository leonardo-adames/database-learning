# Otras extensiones del modelo E/R: [Dinámica Del [MER]]

## Dinámica del modelo E/R**

### ¿Qué es la dinámica del modelo E/R?

**El modelo E/R tradicional es estático:**

* escribe qué datos existen

* No describe cómo cambian en el tiempo

**La dinámica del modelo E/R introduce:**

* comportamiento, eventos y cambios de estado de las entidades y relaciones.

### ¿Para qué sirve?

**Permite modelar:**

* Evolución temporal de los datos

* Reglas que dependen del tiempo

* Procesos del negocio

## Elementos de la dinámica del modelo E/R

* a) Estados

Una entidad puede tener distintos estados a lo largo del tiempo.

    Ejemplo: Pedido

    Estados: creado → pagado → enviado → entregado → cancelado

* b) Eventos

Un evento provoca un cambio de estado.

**Ejemplo:**

    realizar pago

    enviar pedido

* c) Transiciones

    Son los cambios de un estado a otro, causados por eventos.
